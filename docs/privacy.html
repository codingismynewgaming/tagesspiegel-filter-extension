🔒 Privacy Policy

Last Updated: March 17, 2026

📋 Quick Summary

  • ❌ No data collection
  • ❌ No analytics or tracking
  • ❌ No third-party services
  • ✅ All data stored locally in your browser

1. Data Collection

Tagesspiegel Filter does NOT collect, store, or transmit any personal data or user information.

Specifically:

  • No personal information is collected
  • No browsing history is tracked
  • No usage statistics are gathered
  • No data is transmitted to external servers
  • No cookies or similar technologies are used

2. Local Storage

This extension uses your browser's local storage API solely to save your filter preferences. This data:

  • Is stored locally on your device
  • Never leaves your browser
  • Is only accessible by this extension

Firefox Sync Users: If you use Firefox Sync, your preferences may be synchronized across your devices via Mozilla's encrypted sync service. The extension developer has no access to this data.

3. Permissions

This extension requests the following permissions:

Permission Purpose
storage Save your filter preferences locally
activeTab Apply filters to the current Tagesspiegel.de tab
https://www.tagesspiegel.de/* Modify page content to hide selected sections

4. Third-Party Services

This extension does not integrate with any third-party services:

  • No analytics platforms (e.g., Google Analytics)
  • No advertising networks
  • No tracking or telemetry services
  • No data brokers

5. Data Security

All extension functionality runs locally within your browser. No data is transmitted over the network by this extension.

6. Changes to This Policy

Any changes to this privacy policy will be documented in the extension's changelog and version history.
