## German Version (Deutsch)

### Short Description (Max 130 characters)

### Full Description
**For the detailed description field**

```
Der Tagesspiegel Filter ist eine kleine, aber feine Browser-Erweiterung, die dir dabei hilft, dein Leseerlebnis auf Tagesspiegel.de zu personalisieren.

🎯 Was kann die Erweiterung?

✅ Sektionen ausblenden: Wähle über ein einfaches Menü aus, welche Themenbereiche (z.B. Berlin, Sport, Wirtschaft, Kultur, Politik) du nicht sehen möchtest.

✅ Sauberes Design: Die ausgeblendeten Bereiche verschwinden komplett - deine Startseite bleibt übersichtlich.

✅ Deine Einstellungen werden gespeichert: Die Erweiterung merkt sich deine Auswahl automatisch - auch beim nächsten Besuch.

✅ Privatsphäre: Keine Werbung, kein Tracking, keine Datensammlung. Alles passiert nur in deinem Browser.

📊 Features im Überblick:
• Einfache Checkbox-Oberfläche
• Echtzeit-Filterung
• Statistiken über ausgeblendete Sektionen
• Automatische Speicherung deiner Einstellungen
• Open Source und transparent

🔒 Datenschutz:
Die Erweiterung sammelt keine persönlichen Daten, speichert nichts außerhalb des lokalen Browserspeichers und überträgt keine Daten an Dritte.

Viel Spaß beim personalisierten Lesen! 📰✨

---
Dieses Projekt ist eine inoffizielle Browser-Erweiterung und steht in keiner Verbindung zu Tagesspiegel, R. Pieper Verlag GmbH oder deren Partnern.
```

## English Version

### Short Description (Max 130 characters)
**For the store listing summary**

```
Hide unwanted topics on Tagesspiegel.de - personalize your news feed with one click. Free, secure, open source.
```

**Alternative (shorter):**
```
Hide unwanted sections on Tagesspiegel.de - customize your reading experience with one click.
```

### Full Description
**For the detailed description field**

```
Tagesspiegel Filter is a simple yet powerful browser extension that helps you personalize your reading experience on Tagesspiegel.de.

🎯 What does it do?

✅ Hide sections: Choose which topics you're not interested in (e.g., Berlin, Sports, Business, Culture, Politics) with a simple menu.

✅ Clean design: Hidden sections disappear completely - your homepage stays clean and focused.

✅ Your settings are saved: The extension remembers your preferences automatically - even on your next visit.

✅ Privacy first: No ads, no tracking, no data collection. Everything happens only in your browser.

📊 Features:
• Simple checkbox interface
• Real-time filtering
• Statistics about hidden sections
• Automatic settings synchronization
• Open source and transparent

🔒 Privacy:
The extension doesn't collect any personal data, stores nothing outside local browser storage, and doesn't transmit data to third parties.

Enjoy personalized reading! 📰✨

---
This project is an unofficial browser extension and is not affiliated with Tagesspiegel, R. Pieper Verlag GmbH or their partners.
```
